# Function to extract TFFF colors as hex codes

tfff_colors() enables you to pull colors directly from the tfff palette.
Choose one of the following options: \`Dark Green\`, \`Light Green\`,
\`Orange\`, \`Yellow\`, \`Blue\`, \`Red\`, \`Brown\`, \`Dark Gray\`,
\`Medium Gray\`, \`Light Gray\`.

## Usage

``` r
tfff_colors(...)
```

## Arguments

- ...:

  color or colors you want to retun

## Value

A vector of colors
